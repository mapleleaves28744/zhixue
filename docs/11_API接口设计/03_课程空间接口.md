# 03_课程空间接口

> 用途：课程创建、课程列表、课程详情、课程删除接口。

## 编写状态

- [x] 已补充基础 CRUD 接口

## 接口清单

| 功能 | 方法 | 路径 | 权限 | 说明 |
|---|---|---|---|---|
| 创建课程 | POST | `/api/v1/courses` | Student/Admin | 学生创建自己的课程空间；非管理员固定为 `private` |
| 课程列表 | GET | `/api/v1/courses` | Student/Admin | 学生仅返回自己的课程，管理员返回全部 |
| 课程详情 | GET | `/api/v1/courses/{course_id}` | Owner/Admin | 无权限时按不存在处理 |
| 更新课程 | PUT | `/api/v1/courses/{course_id}` | Owner/Admin | 学生不可修改 `visibility` |
| 归档课程 | DELETE | `/api/v1/courses/{course_id}` | Owner/Admin | 软删除，设置 `status=archived` |

## 查询参数

`GET /api/v1/courses` 支持：

- `page`：页码，默认 1。
- `page_size`：每页数量，默认 20，最大 100。
- `status`：`active` / `archived` / `all`，默认 `active`。

## 请求与响应

创建课程请求体：

```json
{
  "title": "数据结构",
  "course_code": "CS-DS-001",
  "description": "计算机专业核心课程",
  "subject": "计算机科学与技术",
  "visibility": "private"
}
```

课程对象字段：

```json
{
  "id": "uuid",
  "owner_id": "uuid",
  "title": "数据结构",
  "course_code": "CS-DS-001",
  "description": "计算机专业核心课程",
  "subject": "计算机科学与技术",
  "cover_url": null,
  "visibility": "private",
  "status": "active",
  "created_at": "2026-05-22T00:00:00Z",
  "updated_at": "2026-05-22T00:00:00Z"
}
```

所有接口使用统一响应结构：`code`、`message`、`data`、`request_id`。
