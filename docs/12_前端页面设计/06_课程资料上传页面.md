# 06_课程资料上传页面

> 用途：设计资料上传、解析进度、文件管理。

## 编写状态

- [x] 已补充当前实现说明

## 页面入口

当前功能接入 authenticated 学生端：

- `/student/dashboard`：课程列表与创建课程入口。
- `/student/wiki`：课程资料上传、资料列表、解析状态与手动解析入口。

公开展示页 `/courses`、`/knowledge` 仍保留 Stitch 静态页面，不在本阶段替换。

## 组件

| 组件 | 位置 | 说明 |
|---|---|---|
| `CoursePageShell` | `frontend/components/course` | 学生端课程列表和创建入口 |
| `CourseSelector` | `frontend/components/material` | 选择当前课程 |
| `FileUploader` | `frontend/components/material` | 上传 PDF/DOCX/MD/TXT |
| `MaterialList` | `frontend/components/material` | 展示资料、解析状态、解析按钮 |

## 视觉要求

页面沿用 Luminous Warmth：

- 暖浅背景 `#fcf9f8`。
- 玻璃拟态卡片、柔和阴影、大圆角。
- 主色 `#835400`、`#f9a826`。
- Material Symbols 图标体系。
- AI/Wiki 相关文案必须强调来源、解析状态和后续可追溯链路。
