# 数据结构课程知识库 / 知识图谱 / LLM Wiki 资料包

生成日期：2026-05-22

本资料包用于构建“数据结构”课程的：
- 知识库：面向 RAG 检索的 JSONL 切片；
- 知识图谱：实体、关系、三元组 CSV；
- LLM Wiki：Markdown 风格结构化知识页；
- 测评系统：选择题、简答题、编程题、诊断题；
- 学习路径：先修关系、模块路径、难度梯度；
- 导入模板：适合导入数据库、向量库、图数据库或自研 LLM Wiki。

## 目录说明

1. `01_课程体系与教学大纲`：课程目标、章节结构、知识点目录。
2. `02_LLMWiki知识页`：每个模块一份 Wiki 页面，包含定义、结构、算法、复杂度、易错点。
3. `03_知识库切片_jsonl`：适合向量化入库的分块数据，每条包含 chunk_id、title、module、concepts、content、tags、difficulty。
4. `04_知识图谱_csv`：实体表、关系表、三元组表，可导入 Neo4j / Gephi / 自研图谱服务。
5. `05_题库与测评`：分层题库、答案、解析和测评标签。
6. `06_代码示例`：常见数据结构与算法的 Python 示例。
7. `07_学习路径与先修关系`：推荐学习路线、先修依赖、补弱策略。
8. `08_检索增强与提示词模板`：RAG 问答、图谱推理、个性化学习提示词。
9. `09_导入模板`：数据库建表建议、Neo4j 导入 Cypher、LLM Wiki 页面字段模板。

## 使用建议

- 构建向量知识库：优先使用 `03_知识库切片_jsonl/ds_kb_chunks.jsonl`。
- 构建知识图谱：使用 `04_知识图谱_csv/entities.csv`、`relations.csv`、`triples.csv`。
- 构建 LLM Wiki：使用 `02_LLMWiki知识页` 下的 Markdown 文件。
- 构建测评系统：使用 `05_题库与测评/question_bank.jsonl`。
- 做个性化推荐：结合 `07_学习路径与先修关系/prerequisite_edges.csv` 与题库掌握标签。

## 注意

这些内容是原创整理型教学资料，不是对某本教材或网页的复制。适合课程知识库原型、比赛项目 Demo、LLM Wiki 初始语料与知识图谱冷启动。
