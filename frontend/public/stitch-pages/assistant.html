请解释一下洛必达法则的使用条件，并给出一个简单的例子。

auto_awesome

洛必达法则用于求解未定式极限，主要是“0/0”型或“∞/∞”型。使用条件如下：

  • 分子分母趋于 0 或无穷大。
  • 在趋于某一点的去心邻域内，两者都可导。
  • 分母的导数不为 0。
  • 求导后的极限存在（或为无穷大）。

示例：求 lim(x->0) (sin x) / x

这是一个 0/0 型极限。
f(x) = sin x, f'(x) = cos x
g(x) = x, g'(x) = 1
lim(x->0) (cos x) / 1 = cos(0) / 1 = 1
